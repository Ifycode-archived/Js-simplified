INTRODUCTION

Here you will find the background needed to follow through with the content of all other chapters: HTML structure, the DOM, how to create & select elements and the basics for styling elements on the webpage. I'll begin by introducing HTML. HTML = Hyper Text Markup Language. Markup Language makes use of tags to create and/or identify content on the webpage. See definition of webpage and some other terms here. Text editor is used to write or edit markup language and programming language. See difference between the languages and also read about scripting language here. You can start with the editor already on your computer e.g. Notepad, but as you grow, there will need to download and install better editors like vs code, sublime text etc. Html files are saved with .html (or .htm) file extension.

HTML STRUCTURE

DOM = Document Object Model. The DOM presents the HTML document like a tree. We'll now introduce the tags one after the other to understand this tree-like relationship between elements on the page better. The html tag houses two other tags. Add the tags following these steps: Step 1: Add the html tag <html></html> Step 2: Add the head tag <html> <head></head> </html> Step 3: Add the body tag <html> <head></head> <body></body> </html> Let's call our attention to important knowledge/tips we will be needing later on. html is a parent element and has two child elements: head and body. Note: there can only be one html tag in the html document. Inside the head tag: <html> <head> <title> Page title here </title> </head> <body></body> </html> Inside the body tag: <html> <head> <title> Page title here </title> </head> <body> <p> I am a paragraph </p> </body> </html> Head and body now have one child element each: title and paragraph respectively (p for paragraph tag). The tags introduced so far have opening tag <tagNameHere> and closing tag </tagNameHere>. Note: there can only be one head tag and one body tag in the html document. We will now cover some more of tags that can be found in the head and/or in the body.

ADDING JAVASCRIPT

The script tag - Before now, the script tag would look like this; <script language="Javascript" type="text/javascript"></script> The language and type attributes are no longer used/required since javascript is (now) the default HTML scripting language. Attributes are placed in the opening tag to provide additional information about the element in question. They are written in name and value pairs: name="value". The script tag src attribute: Instead of writing code in-between script tags, you can use external javascript i.e. write code in a separate file, save it with .js file extension and reference it in the html document using the src attribute. <script src="nameOfJSfile.js"></script> Create a separate js file in the same folder where you have your index.html file (html file for homepage is usually given the name index.html). Add js file name in this manner: <script src="alertTest.js"></script> To be more organised, instead create a folder (named js) for your external js file(s) in the same folder where you have your index.html file. Then put alertTest.js file in it. Reference it in the html like this: <script src="folderNameHere/nameOfJSfile.js"></script>

With external javascript, we are able to separate js from html - code is easier to read. Other advantages are; we can reuse javascript code just by referencing in html and page loads faster. Preferably, place script tag in the body.
