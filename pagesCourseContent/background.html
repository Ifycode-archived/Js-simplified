INTRODUCTION

Here you will find the background needed to follow through with the content of all other chapters: HTML structure, the DOM, how to create & select elements and the basics for styling elements on the webpage. I'll begin by introducing HTML. HTML = Hyper Text Markup Language. Markup Language makes use of tags to create and/or identify content on the webpage. See definition of webpage and some other terms here. Text editor is used to write or edit markup language and programming language. See difference between the languages and also read about scripting language here. You can start with the editor already on your computer e.g. Notepad, but as you grow, there will need to download and install better editors like vs code, sublime text etc. Html files are saved with .html (or .htm) file extension.

HTML STRUCTURE

The structure of HTML document can be likened to a family tree. Relationship between elements on the page is described with terms like ancestors, decendants, parent, children and siblings. We'll now introduce the tags one after the other to understand this tree-like relationship between elements on the page better. The html tag houses two other tags. Add the tags following these steps: Step 1: Add the html tag <html></html> Step 2: Add the head tag <html> <head></head> </html> Step 3: Add the body tag <html> <head></head> <body></body> </html> html is a parent element and has two child elements: head and body. Head and body are siblings. Note: there can only be one html tag in the html document. The html element is the root of the html document. Inside the head tag: <html> <head> <title> Page title here </title> </head> <body></body> </html> Inside the body tag: <html> <head> <title> Page title here </title> </head> <body> <p> I am a paragraph </p> </body> </html> Head and body now have one child element each: title and paragraph respectively (p for paragraph tag). The tags introduced so far have opening tag <tagNameHere> and closing tag </tagNameHere>. Note: there can only be one head tag and one body tag in the html document. We will now cover some more of tags that can be found in the head and/or in the body. Comments: In HTML, anything written in-between <!-- --> is a comment. You will soon see how comments work in subsequent examples.

SELECTING ELEMENTS (JAVASCRIPT)

In javascript we can select elements by tag name, class name and by Id not just for styling purpose, but also to add or remove elements on the page. We access elements on the webpage through document object i.e. the root of all objects on the page. By 'all objects on the page' i mean 'all elements on the page'. Objects usually have methods: objectName.methodName(); The document object has methods with which we are able to select elements:

document.getElementsByTagName("tagNameHere"); document.getElementsByClassName("classNameHere"); document.getElementById("IdNameHere");

We mentioned that Id is unique earlier, this is reflected in the method name 'getElement'. TagName and class name on the other hand use 'getElements'. Selecting elements by Id is straight forward. To be able to select one out of many elements with the same tag name or class name, we need the knowledge of a type of object called array. Class name method store elements on the page that have the same class name as an array, while tag name method store sibling elements on the page that have the same tag name as an array. Think of array as something that stores a list of items and whose elements are accessed using index numbers. The index numbers are enclosed in square brackets. The first item has index number 0, the second has index number 1 etc. So if there are 3 sibling div elements inside the body: <body> <div></div> <div></div> <div></div> </body> Selecting 1st div: document.getElementsByTagName("div")[0]; Selecting 2nd div: document.getElementsByTagName("div")[1]; Selecting 3rd div: document.getElementsByTagName("div")[2]; Styling elements with javascript: selectedElement.style.property = "value"; Note: properties written with dash in it in CSS e.g. margin-left are written with camelCase in javascript.
